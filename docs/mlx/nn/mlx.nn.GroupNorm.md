Computes the same normalization as layer norm, namely \\\\\\\\\\\\\\\\y = \\\\\\\\frac{x - E\\\\\\\\\\\\\\\[x\\\\\\\\\\\\\\\]}{\\\\\\\\sqrt{Var\\\\\\\\\\\\\\\[x\\\\\\\\\\\\\\\]} + \\\\\\\\epsilon} \\\\\\\\gamma + \\\\\\\\beta,\\\\\\\\\\\\\\\\ where \\\\\\\\\\\\\\\\\\\\\\\\gamma\\\\\\\\\\\\\\\\ and \\\\\\\\\\\\\\\\\\\\\\\\beta\\\\\\\\\\\\\\\\ are learned per feature dimension parameters initialized at 1 and 0 respectively.
